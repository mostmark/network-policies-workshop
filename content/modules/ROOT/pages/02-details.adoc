== Troubleshooting guide

=== Common setup issues

**Problem**: "oc command not found"

**Solution**: Make sure you are running commands in the Web Terminal in the OpenShift console (click the *>_* icon in the upper-right masthead). The `oc` CLI is preinstalled in the Web Terminal. If you are using a local terminal, switch to the Web Terminal instead.

**Problem**: "Unable to connect to OpenShift cluster"

**Solution**: Verify network connectivity and login credentials. Try logging in again:

[source,bash]
----
oc login -u %OPENSHIFT_USERNAME% -p %OPENSHIFT_PASSWORD% %OPENSHIFT_API_URL%
----

**Problem**: "Permission denied" when creating resources

**Solution**: Verify you're logged in as the correct user and have the necessary permissions:

[source,bash]
----
oc whoami
oc auth can-i create networkpolicies
----
